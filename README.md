Создал проект "gabiden_practice" для работы с пакетами. 
Проект содержит пакет `powerbill`, который позволяет рассчитывать потребление электроэнергии, стоимость и формировать отчет.
Пакет спроектирован так, чтобы его можно было использовать в других проектах как внешний модуль.
- `pkg/powerbill` — пакет с экспортируемыми функциями  
- `cmd/app/main.go` — пример использования пакета  
- `go.mod` — модуль проекта
Использованы функции: Consumption, EnergyCost, ApplyDiscount, FormatEnergyReport. Все функции проверяют входные данные и возвращают ошибки через `fmt.Errorf()`.
